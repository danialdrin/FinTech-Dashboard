# 💎 Financial Atelier — Premium Fintech Dashboard

**Financial Atelier** is a state-of-the-art wealth management interface built with **React, Vite, and Framer Motion**. It replaces traditional "box-and-border" layouts with a sophisticated **"No-Line" Editorial Design System**, utilizing tonal layering and motion to guide the user's focus through complex fiscal data.

---

## 🚀 Key Features

### 1. **Unified Wealth Portfolio**
- **Real-Time Stats**: Instant visibility into Global Portfolio, Monthly Income, and Liabilities.
- **Visual Growth Tracking**: High-fidelity `AreaChart` with gradient fill tracking quarterly prosperity.

### 2. **Operational Ledger (Transactions)**
- **Symmetrical Alignment**: Pillar-locked columns for Category and Status indicators.
- **Pixel-Perfect Centering**: Status and Category pills are horizontally centered within their column "boxes" for high-speed scanning.
- **Robust Truncation**: Automatic merchant name handling to ensure layout integrity on all resolutions.

### 3. **Fiscal Insights & Predictive Analytics**
- **Cashflow Pulse**: Dual-value `BarChart` comparing operational income versus overheads.
- **Wealth Forecasting**: AI-style predictive descriptions based on current spending velocity and liquidity.

### 4. **Interactive Command Modals**
Premium, glassmorphic overlays for critical financial operations:
- **Send**: Quick peer-to-peer transfer logic.
- **Add**: Simplified transaction logging with tonal inputs.
- **Cards**: Digital card visualization with management toggles.
- **Plan**: Real-time progress monitoring for preservation goals.

## 🛠 Engineering Stack

| Stack | Technology | Purpose |
| :--- | :--- | :--- |
| **Foundation** | React 18 + Vite | Rapid HMR and component-based state management. |
| **Motion** | Framer Motion | AnimatePresence-driven modals and staggered list entries. |
| **Visualization** | Recharts | Modular, accessible SVG-based data rendering. |
| **Styling** | Vanilla CSS3 | Custom Design Tokens utilizing tonal surface levels (v1-v5). |
| **Icons** | Lucide React | Minimalist, stroke-based iconography. |

---

## 🎨 Design Philosophy: "The Editorial Rule"

This dashboard eliminates traditional borders. Instead, depth and hierarchy are achieved through **Tonal Layering**:
- **Layer 0**: Main surface background.
- **Layer 1-3**: Stat cards and secondary panels using tonal shifts.
- **Layer 5**: Active overlays, tooltips, and floating modals.
